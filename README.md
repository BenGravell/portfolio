Feel free to connect with me on [GitHub](https://github.com/BenGravell) or [LinkedIn](https://www.linkedin.com/in/benjamin-gravell/).

## Development

Run locally with

```bash
bundle exec jekyll serve
```

Install dependencies with

```bash
bundle
```
